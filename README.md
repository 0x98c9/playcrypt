# Playcrypt

**Playcrypt** is a fun, privacy-focused web app that lets you encrypt and decrypt messages using emojis! Instantly turn your text into a secret emoji code and back again, all in your browser.

![Screenshot](.\public\screenshot.png)

## Features

- **Secure Emoji Encryption:** Converts your text into emojis using a unique mapping system.
- **Instant Conversion:** Real-time, lightning-fast encryption and decryption.
- **One-Click Copy:** Easily copy your emoji messages to share anywhere.
- **Mobile Friendly:** Fully responsive design for all devices.
- **No Data Stored:** All processing happens in your browser—your messages are never sent to a server.

## How It Works

1. **Enter Your Text:** Type or paste the text you want to encrypt.
2. **Convert to Emojis:** Click the convert button to transform your text into emojis.
3. **Decrypt When Needed:** Paste an emoji message to convert it back to text.

## Usage Example

### Encrypting a Message
1. Enter your message in the input box (e.g., `hello world`).
2. Click **Convert** (with Encrypt mode selected).
3. Copy the emoji output and share it securely!

### Decrypting a Message
1. Paste the emoji message into the input box.
2. Switch to **Decrypt** mode and click **Convert**.
3. Read the original text instantly.

## FAQ

- **Is my encrypted data secure?**  
  Yes! Playcrypt never stores your data. All encryption happens locally.
- **Can I decrypt messages from others?**  
  Yes! Just paste the emoji message into the decrypt section.
- **Is Playcrypt free?**  
  100% free, with no hidden costs.

## Tech Stack

- **React** + **TypeScript**
- **Vite** for fast development
- **Tailwind CSS** for styling
- **Radix UI** for accessible components

## Getting Started

1. **Install dependencies:**
   ```sh
   npm install
   ```
2. **Run the app:**
   ```sh
   npm run dev
   ```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.
